## I Learned..!

### interval과 timeout

interval : 매 간격마다 실행되는 함수

    setInterval(function, number); // number의 단위는 ms

timeout : 특정 시간 후 한번만 실행되는 함수

    setTimeout(function, number); // number의 단위는 ms

### padStart와 padEnd

sting에 string으로 패딩을 추가한다
      
    object.padStart(2, 0); //2는 길이 0은 채워질 string
